    Creating an amendment to the labour contract

    It is possible to create an amendment to the labour contract, when changing the salary, position, loading, etc. To create an amendment, open new labour contract card and place checkmark in the field Alteration. In order to prepare the amendment in more convenient way to the user, it is advised to follow these steps:

    1. Place a checkmark that this is the alteration, and choose the contract number in the field Main Contract No. what you want to alternate; When choosing the main contract, fields Employee No., First Name, Last Name, Social Security No. are entered automatically. If in Personnel Setup is enabled the options By Default Post Order with LC and By Default Create Tax Comp., then after the enabling the field Alteration in the labour contract card these marks disables and system will not allows to enable again Post Order with LC and Create Tax Components.

    2. Choose the amendment type code, from a list Labour Contract Alteration Types, where the following fields are available to enter the types:

    • Code – user freely defined code;
    • Description – Code description;
    • Alteration type – the system has already defined different types of alterations – Position change, Amount change, Add position, Loading change, Close position. By selecting the alteration type, the system performs the necessary control of mandatory field filling. This field is optional and may not be completed;
    • Template code – a document template can be defined at each type of amendment;
    • Mandatory Payroll Component Line – mark if this alteration is required to add also payroll component. In case if checkmark is entered, the system will not allow you to change the status to Completed without entering payroll component line.
    1. Enter Employee No. and choose Employee Position No. from a list, for what alteration need to be made. When choosing Employee Position No., pops out the notification with question - update information from employee position. By pressing Yes, the system will fill in the following fields:
    • Position Code – if this amendment if for to the position change, the new position code is selected in this field;
    • Position Description;
    • Department Code;
    • Department Name;
    • Primary Position – enabled, if alteration made to primary position;
    • Loading;
    • Calendar Code;
    • Calendar Name.

    Complete information from the employee's position card is useful if only one condition is changed, for example, only loading. As well as by choosing to complete the information from the employee's position, the system automatically fills the payroll components, which was defined for a particular employee's position number. The payroll components are described in the next section. By pressing No, the system will not going to fill fields automatically, except Primary Position field, what will be enabled, if alteration made to primary position.

    1. Fill in the other fields:
    • Employment Date – Date when alteration take effect;
    • Planned Termination Date – if the termination date is specified. Informative field.
    • Position Code – a new position code if the position has changed. In other cases, the already existing position code should be selected;
    • Position Description;
    • Department Code – fill in automatically from the position card;
    • Department Name;
    • Date To – if alteration is only for some limited time, then enter the due date in this field. After the end of this date a previous conditions will take effect.

    After the posting of amendment to the labour contract, a personnel order should be issued also.
