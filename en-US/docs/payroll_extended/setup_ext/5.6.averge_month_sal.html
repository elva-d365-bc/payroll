    Monthly average gain calculation

    To calculate the monthly average gain go to:

    • LV Payroll Role Center - Component Registers – + New.
    • in the tab General enter the year, month and description;
    • in the tab Component Register ListPart choose employee from the list, for what you need to calculate the average gain.
    • tap on Line - Calculate Monthly average Gain.

    • system will shows the notification on data for average gain calculation.

    • Press OK and system will calculated the average gain for the month.

    'Monthly average earnings are calculated according to the Labor Law's chapter 75. part 3. requirements (valid from 01.01.2015.). It states that monthly average gain calculation must be performed by daily average earnings multiplied by month average working days in the last six calendar months(counting the working days in the last six calendar months and this amount divide with six). Monthly average working days in the last six calendar months the system is calculating by the following algorithm:

    • If in Employee card's tab Employee Positions are stated Salary Type - normal or aggregated working time – to calculate the average working day count in the last six month, system uses the calendar code what is stated in the Payroll Setup's tab General field Default Calendar.
    • If in Employee card's tab Employee Positions are stated Salary Type - Aggregated working time and Calendar code matches to the code what is stated in the Payroll Setup's field Default Calendar 6DDN, then to calculate the average working day count in the last six month, system uses the calendar code what is stated in the Payroll Setup's tab General field Default Calendar 6DDN.
