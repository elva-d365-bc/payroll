    How to get started

    To start using Dynamics 365 Business Central Latvian Payroll Advanced, there is no need to choose a specific role center. Functionality will work from any role center. However, we recommend to use LV Payroll Role Center, for all payroll shortcuts can be available at home page and the work with payroll would be more convenient and faster.

    Applying Configuration package

    All the basic settings for use of Latvian Payroll Advanced are created by applying the configuration package.

    To apply the configuration pack, click the button Search for the page or report… Then type in the search field Configuration packages and select from the list .

    At the Configuration package window toolbar click on buttons Process - Import Package and select the relevant package file from your file storage. Select the loaded pack in one click and click on the button Process and Apply Package.

    Content of configuration package

    The configuration package enters in the system following information:

    • Number series and the rows
    • Payroll setup
    • Vacation period setup
    • Additional vacation day setup
    • Statistical report setup Work-2
    • Statistical report setup Work-5

    General system setup

    In Company information settings the following fields are mandatory:

    • Company name
    • Registration No.

    In Bank Accounts section with configuration package will be loaded the bank account card for payroll payments.The following fields are mandatory in this card:

    • SWIFT code
    • IBAN

    In Chart of Accounts section with configuration package will be loaded accounts what are essential for the payroll. If necessary, these accounts are recommended to renumber/ renew so that you should not adjust the adjustment settings.

    In Dimensions section should be entered dimensions what will be used for payroll.
