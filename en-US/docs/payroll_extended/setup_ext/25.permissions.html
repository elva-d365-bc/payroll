    Tiesības (algu grāmatvedis un personāla daļa)

    Lai izmantotu Algas un Personāls moduļus, pieejamas sekojošas tiesību kopas:

    Lomas ID Nosaukums Iekļauts Nav iekļauts
    HR-ABSENCE Create and edit absence Rediģēt darbinieku prombūtnes (manuāli un izmantojot Personālu) -
    HR EMPLOYEE Read employees Redzēt visu informāciju par darbinieku, t.s. konfidenciālo un personāla dokumentus Redzēt darbinieka algu datus: algu komponentes un summas no algu sarakstiem
    HR EMPLOYEE, EDIT Edit employees Redzēt visu informāciju par darbinieku, t.s. konfidenciālo un personāla dokumentus. Veidot dokumentu veidnes. Ģenerēt atvaļinājuma periodus. Redzēt darbinieka algu komponentes. Redzēt darbinieka algu sarakstu datus
    HR-HRM-PROTECT Read Protection at Work Redzēt datus par Obligātās veselības pārbaudēm un darba drošības instruktāžām. -
    HR-HRM-PROTECT, EDIT Edit Protection at Work Reģistrēt informāciju par Obligātās veselības pārbaudēm un darba drošības instruktāžām. -
    HR-HRM-SETUP, EDIT Edit Personal Setup Rediģēt visus uzstādījumus no Personāla moduļa, t.sk. ziņojumus, dokumentu veidnes. Veidot amatu vietas un grāmatot amatu rīkojumus. -
    HR PAYROLL Read payroll Redzēt algu sarakstus un algu informāciju darbinieku kartiņās. Redzēt personāla dokumentus un darbinieku personīgo informāciju (Radinieki, Konfidenciāla informācija, Kvalifikācijas, utt.)
    HR PAYROLL, EDIT Edit Payroll Pilns algu aprēķins. Veselības apdrošināšanas reģistrēšana. Datu imports un EDS. -
    HR-PAYROLL SETUP Edit payroll setup Rediģēt visus uzstādījumus no algas un Cilvēku resursu moduļiem. -
    HR-PAYROLL MIGR Payroll migration Algu migrācija -

    Katram lietotājām nepieciešams noradīt tās Tiesību kopas, kas atbilst viņa darba pienākumiem. Pēc vajadzībās, norāda vienu vai vairākas kopas.
