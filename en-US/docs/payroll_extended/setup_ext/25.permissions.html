    Permissions (payroll accountant and HR department)

    In order to use payroll and personnel modules, the following permission sets are available:

    Permission Set ID Name Included Not included
    HR-ABSENCE Create and edit absence Edit employee absences (manually and through personnel module) -
    HR EMPLOYEE Read employees See all the information about the employee, including confidential information and personnel documents See the employee's salary data: payroll components and amounts from salary lists
    HR EMPLOYEE, EDIT Edit employees See all the information about the employee, including confidential information and personnel documents. Create a document templates. Generate vacation periods. See employee payroll components. See the employee's salary list data
    HR-CONFIDENTIAL Create and edit confidential See and edit confidential information (manually and through personnel module). -
    HR-SETUP Human Resources setup Edit all settings related to Payroll and Personnel modules. -

    Each user needs to assign the permission sets that meet his duties. One or more permission sets can be assigned.
